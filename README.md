## Overview

Sanctum Browser is an educational secure browser built for schools and learning institutions. Originally developed by the HCCS Robotics Team as an academic project, it has evolved into a dedicated educational security product under Sanctum Software Development Services.

The web wasn't built for classrooms. Standard browsers weren't designed with students in mind — every tab, every search, every distraction was someone else's feature. Sanctum Browser was built from scratch to fix that.

## Features

- **Site Whitelisting & Blacklisting** — Only approved educational sites are accessible. Everything else is blocked by default.
- **Exam Mode** — Locks students into specific sites during tests. Full lockdown with no way out.
- **Content Filtering** — Intercepts network requests to block URLs before they load.
- **Real-Time Monitoring** — Tab monitoring detects switching away from allowed sites. Activity logged for teacher review.
- **Zero Telemetry** — No tracking, no ads, no data sold to third parties.
- **Lightweight** — Runs well on underpowered school laptops. No hardware upgrades required.

## How It Works

| Mode | Description |
|------|-------------|
| **Whitelist & Blacklist** | Configurable per school policy. Allows Google Classroom, Khan Academy, Wikipedia, and more. Blocks AI tools, social media, games, and streaming. |
| **Exam Mode** | Restricts access to only exam-related sites. Tab switching disabled. Tamper-proof, can't be bypassed. |
| **Content Filtering** | Blocks harmful content before it loads. Real-time URL interception. |

## Trust

- **Classroom-First Design** — Built by someone sitting in those classrooms, not reverse-engineered from a corporate IT product.
